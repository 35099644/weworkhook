# weworkhook-android
企业微信打卡地址修改，支持手动输入经纬度或地图选点。
[release](url 'https://github.com/gallonyin/weworkhook/releases')页面有apk安装包可以直接下载使用，安装后在xposed里勾选并重启设备即可。
作者尝试非 ROOT 设备使用 virtualxposed 软件同样可以生效，没有 ROOT 的朋友可以自行尝试。
